## 🌟 What is PRECIOUS-MD?

> **PRECIOUS-MD** is a next-generation, feature-rich WhatsApp bot built with **Node.js** and **Baileys**. It's not just a bot — it's a full automation platform with multi-session support, AI superpowers, and a live web dashboard. Designed for power users, group admins, and developers who want the best WhatsApp automation experience.

## 🚀 Deployment Guide

### 1️⃣ Prerequisites

- ✅ Node.js 18+
- ✅ MongoDB Atlas (free tier works fine)
- ✅ Git installed
- ✅ Heroku / Railway / VPS account

### 2️⃣ Environment Variables

Create a `.env` file:

```env
# ─── Required ─────────────────────────────────
MONGODB_URI=mongodb+srv://<user>:<pass>@cluster0.mongodb.net/precious_md
PORT=3000
OWNER_NUMBER=923216046022

# ─── Dashboard ────────────────────────────────
DASHBOARD_PASSWORD=your_strong_password_here

# ─── AI Features ──────────────────────────────
GEMINI_API_KEY=your_gemini_api_key_here

# ─── Optional ─────────────────────────────────
BOT_NAME=PRECIOUS-MD
PREFIX=.
```

> ⚠️ **Never commit your `.env` file to GitHub!** Add it to `.gitignore`.

### 3️⃣ Heroku Deployment

```bash
# 1. Clone the repo
git clone https://github.com/Mudassar-khan-M2K/precious-md
cd precious-md

# 2. Install dependencies
npm install

# 3. Test locally first
npm start
# → Open http://localhost:3000

# 4. Deploy to Heroku
heroku create your-bot-name
heroku config:set MONGODB_URI=your_mongodb_uri
heroku config:set DASHBOARD_PASSWORD=your_password
heroku config:set GEMINI_API_KEY=your_key
heroku config:set OWNER_NUMBER=923216046022

git add .
git commit -m "🚀 Deploy PRECIOUS-MD v3.0.0"
git push heroku main
heroku ps:scale web=1
heroku open
```

### 4️⃣ First Login

```
1. Open your app URL
2. Enter your dashboard password
3. Click "Add Session"
4. Enter your WhatsApp number
5. Choose QR Code or Pairing Code
6. Scan / enter code in WhatsApp
7. Done! Send .menu to test 🎉
```

## ⚠️ Important Notes

> 🔴 **Anti-Ban**: Avoid excessive multi-session reactions/polls. The built-in delay system handles this automatically — don't disable it.

> 🟡 **Instagram**: May require cookies for private content. Public content works out-of-the-box.

> 🟢 **Heroku**: Recommended max **15 sessions** on a basic dyno for stable performance.

> 🔵 **Session Lost?** Check your MongoDB connection string and make sure the Atlas cluster is accessible.
